IDs imprevisíveis
Se o ID não puder ser detectado usando os métodos acima, um excelente método de detecção de IDOR é criar duas contas e trocar os números de ID entre elas. Se você conseguir visualizar o conteúdo de outros usuários usando o número de ID deles enquanto estiver conectado com uma conta diferente (ou mesmo sem estar conectado), você encontrou uma vulnerabilidade IDOR válida.
