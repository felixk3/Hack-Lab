IDs codificados
Ao transmitir dados de uma página para outra, seja por meio de requisições POST, strings de consulta ou cookies, os desenvolvedores web geralmente primeiro pegam os dados brutos e os codificam. A codificação garante que o servidor web receptor consiga entender o conteúdo. Ela transforma dados binários em uma string ASCII, geralmente usando o a-z, A-Z, 0-9 and = caractere `\` para preenchimento. A técnica de codificação mais comum na web é a codificação Base64, que geralmente é fácil de identificar. Você pode usar sites como https://www.base64decode.org/ para decodificar a string, editar os dados e recodificá-los usando https://www.base64encode.org/ e, em seguida, reenviar a requisição web para verificar se houve alguma alteração na resposta.
