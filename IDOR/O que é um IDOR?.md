O que é um IDOR ?

IDOR significa Referência Direta Insegura a Objetos e é um tipo de vulnerabilidade de controle de acesso.

Esse tipo de vulnerabilidade pode ocorrer quando um servidor web recebe dados fornecidos pelo usuário para recuperar objetos (arquivos, dados, documentos), deposita-se muita confiança nesses dados e eles não são validados no servidor para confirmar se o objeto solicitado pertence ao usuário que o fez.
