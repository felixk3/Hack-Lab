IDs criptografados
Os IDs criptografados (hash) são um pouco mais complicados de lidar do que os codificados, mas podem seguir um padrão previsível, como ser a versão criptografada do valor inteiro. Por exemplo, o número de ID 123 se tornaria 202cb962ac59075b964b07152d234b70 se o hash MD5 estivesse sendo usado.



Vale a pena usar um serviço online como o https://crackstation.net/ (que possui um banco de dados com bilhões de resultados de hash para valor) para verificar se encontramos alguma correspondência.
