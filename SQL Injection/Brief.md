A Injeção de SQL (Structured Query Language), também conhecida como SQLi , é um ataque a um servidor de banco de dados de uma aplicação web que causa a execução de consultas maliciosas. Quando uma aplicação web se comunica com um banco de dados usando dados de um usuário que não foram devidamente validados, existe a possibilidade de um atacante roubar, excluir ou alterar dados privados e de clientes, além de atacar os métodos de autenticação da aplicação web em áreas privadas ou de clientes. É por isso que a SQLi é uma das vulnerabilidades mais antigas em aplicações web e também uma das mais prejudiciais.

Nesta aula, você aprenderá o que são bancos de dados, o que é SQL com alguns comandos básicos de SQL , como detectar vulnerabilidades de SQL , como explorar vulnerabilidades de SQLi e, como desenvolvedor, como você pode se proteger contra a Injeção de SQL .
