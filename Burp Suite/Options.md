Antes de mergulharmos no Burp Proxy , vamos explorar as opções disponíveis para configurar o Burp Suite . Existem dois tipos de configurações: Configurações globais (também conhecidas como configurações do usuário) e Configurações do projeto.

Configurações Globais : Essas configurações afetam toda a instalação do Burp Suite e são aplicadas sempre que você inicia o aplicativo. Elas fornecem uma configuração básica para o seu ambiente Burp Suite .

Configurações do projeto : Essas configurações são específicas para o projeto atual e se aplicam somente durante a sessão. No entanto, observe que o Burp Suite Community Edition não permite salvar projetos; portanto, quaisquer opções específicas do projeto serão perdidas ao fechar o Burp.

Para acessar as configurações, clique no botão Configurações na barra de navegação superior. Isso abrirá uma janela de configurações separada.

Janela de configurações separada

Na janela Configurações, você encontrará um menu no lado esquerdo. Este menu permite alternar entre diferentes tipos de configurações, incluindo:

Pesquisa : Permite pesquisar configurações específicas usando palavras-chave.
Filtro de tipo : Filtra as configurações para opções de Usuário e Projeto  .
Configurações do usuário : Exibe as configurações que afetam toda a instalação do Burp Suite .
Configurações do projeto : Exibe as configurações específicas do projeto atual.
Categorias : Permite selecionar configurações por categoria.
Botão de configurações do projeto

Vale ressaltar que muitas ferramentas do Burp Suite oferecem atalhos para categorias específicas de configurações. Por exemplo, o módulo Proxy inclui um botão de configurações de proxy que abre a janela de configurações diretamente na seção de proxy relevante .

Botão de configurações de proxy que abre a janela de configurações diretamente na seção de proxy relevante.

A função de busca na página de configurações é um recurso valioso, permitindo que você encontre rapidamente as configurações desejadas usando palavras-chave.

Dedique algum tempo para se familiarizar com a gama de opções configuráveis ​​do Burp Suite . Assim que se sentir confortável, poderá prosseguir com os exercícios relacionados à configuração das definições do Burp Suite .
