Embora o Burp Suite Community ofereça um conjunto de recursos mais limitado em comparação com a edição Professional, ele ainda fornece uma gama impressionante de ferramentas que são extremamente valiosas para testes de aplicações web. Vamos explorar alguns dos principais recursos:

Proxy : O Burp Proxy é o recurso mais renomado do Burp Suite . Ele permite a interceptação e modificação de requisições e respostas durante a interação com aplicações web.
Repetidor : Outro recurso bastante conhecido.  O Repetidor permite capturar, modificar e reenviar a mesma solicitação várias vezes. Essa funcionalidade é particularmente útil ao criar payloads por tentativa e erro (por exemplo, em SQLi - Injeção de Linguagem de Consulta Estruturada) ou ao testar a funcionalidade de um endpoint em busca de vulnerabilidades.
Intruder : Apesar das limitações de taxa no Burp Suite Community, o Intruder permite bombardear endpoints com requisições. É comumente utilizado para ataques de força bruta ou fuzzing de endpoints.
Decodificador : O decodificador oferece um serviço valioso para transformação de dados. Ele pode decodificar informações capturadas ou codificar cargas úteis antes de enviá-las ao destino. Embora existam serviços alternativos para essa finalidade, utilizar o decodificador dentro do Burp Suite pode ser altamente eficiente.
Comparador : Como o nome sugere, o Comparador permite a comparação de dois conjuntos de dados em nível de palavra ou de byte. Embora não seja exclusivo do Burp Suite , a capacidade de enviar segmentos de dados potencialmente grandes diretamente para uma ferramenta de comparação com um único atalho de teclado acelera significativamente o processo.
Sequenciador : O sequenciador é normalmente empregado para avaliar a aleatoriedade de tokens, como valores de cookies de sessão ou outros dados supostamente gerados aleatoriamente. Se o algoritmo usado para gerar esses valores não garantir aleatoriedade segura, pode expor brechas para ataques devastadores.
Além dos recursos integrados, a base de código Java do Burp Suite facilita o desenvolvimento de extensões para aprimorar a funcionalidade da estrutura. Essas extensões podem ser escritas em Java, Python (usando o interpretador Java Jython) ou Ruby (usando o interpretador Java JRuby). O módulo Burp Suite Extender permite o carregamento rápido e fácil de extensões na estrutura, enquanto o marketplace, conhecido como BApp Store , possibilita o download de módulos de terceiros. Embora algumas extensões possam exigir uma licença profissional para integração, ainda existe um número considerável de extensões disponíveis para o Burp Community. Por exemplo, o módulo Logger++ pode estender a funcionalidade de registro (logging) integrada do Burp Suite .
