Baldes S3

Os buckets do S3 são um serviço de armazenamento fornecido pela Amazon AWS , que permite salvar arquivos e até mesmo conteúdo estático de sites na nuvem, acessíveis via HTTP e HTTPS. O proprietário dos arquivos pode definir permissões de acesso para torná-los públicos, privados e até mesmo graváveis. Às vezes, essas permissões de acesso são configuradas incorretamente e, inadvertidamente, permitem o acesso a arquivos que não deveriam estar disponíveis ao público. O formato dos buckets do S3 é http (s):// {nome}.s3.amazonaws.com, onde {  nome} é definido pelo proprietário, como  tryhackme-assets.s3.amazonaws.com. Os buckets do S3 podem ser descobertos de várias maneiras, como encontrando os URLs no código-fonte da página do site, em repositórios do GitHub ou até mesmo automatizando o processo. Um método comum de automação é usar o nome da empresa seguido por termos comuns, como  {nome} -assets,  {nome} -www,  {nome} -public,  {nome} -private, etc.
