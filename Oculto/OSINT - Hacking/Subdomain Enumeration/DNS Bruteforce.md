A enumeração de DNS (Sistema de Nomes de Domínio) por força bruta é o método de tentar dezenas, centenas, milhares ou até milhões de subdomínios diferentes a partir de uma lista predefinida de subdomínios comumente usados. Como esse método requer muitas solicitações, nós o automatizamos com ferramentas para tornar o processo mais rápido. Neste caso, estamos usando uma ferramenta chamada dnsrecon para realizar essa tarefa. Clique no botão "Visualizar Site" para abrir o site estático, pressione o botão "Executar Solicitação DNSrecon" para iniciar a simulação e, em seguida, responda à pergunta abaixo.
