Quando um certificado SSL/ TLS (Secure Sockets Layer/Transport Layer Security) é criado para um domínio por uma Autoridade Certificadora ( AC ), as ACs participam do que é chamado de "logs de Transparência de Certificados (CT)". Esses logs são publicamente acessíveis e contêm informações sobre todos os certificados SSL/ TLS criados para um nome de domínio. O objetivo dos logs de Transparência de Certificados é impedir o uso de certificados maliciosos ou criados acidentalmente. Podemos usar esse serviço a nosso favor para descobrir subdomínios pertencentes a um domínio. Sites como  https://crt.sh oferecem um banco de dados pesquisável de certificados que exibe resultados atuais e históricos.
