GitHub

Para entender o GitHub, primeiro você precisa entender o Git . O Git é um  sistema de controle de versão  que rastreia as alterações feitas nos arquivos de um projeto. Trabalhar em equipe fica mais fácil porque você consegue ver o que cada membro da equipe está editando e quais alterações foram feitas nos arquivos. Quando os usuários terminam de fazer suas alterações, eles as confirmam com uma mensagem e, em seguida, as enviam de volta para um local central (repositório) para que os outros usuários possam baixar essas alterações para suas máquinas locais. O GitHub é uma versão hospedada do Git na internet. Os repositórios podem ser públicos ou privados e ter vários controles de acesso. Você pode usar a ferramenta de busca do GitHub para procurar por nomes de empresas ou sites para tentar localizar repositórios pertencentes ao seu alvo. Uma vez encontrados, você pode ter acesso ao código-fonte, senhas ou outros conteúdos que ainda não havia encontrado.
